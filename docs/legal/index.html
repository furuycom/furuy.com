Privacy#

furuy.com is a personal blog. No ads or tracking tools are used.

What data may be processed?#

  • Server logs: The site is hosted on GitHub Pages. The hosting/CDN layer may generate technical logs (IP address, request time, requested URL, user agent, etc.) for security and performance purposes.

Third-party services#

  • Comments (Giscus / GitHub Discussions): Comments are provided via Giscus and backed by GitHub Discussions. Loading the comments section sends requests to Giscus and GitHub infrastructure; posting may require a GitHub account and authorization.
    • Giscus Privacy Policy
    • GitHub Privacy Policies

Cookies and similar technologies#

This website does not use advertising or tracking cookies.

  • Preferences (browser storage): User preferences (such as theme) may be stored in browser storage (e.g., localStorage key pref-theme). This information stays on your device and is not sent to the site server.
  • Comments: Loading comments may involve cookies or similar storage mechanisms from Giscus/GitHub.
