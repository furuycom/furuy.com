Gizlilik#

furuy.com kişisel bir blog sitesidir. Reklam veya izleme amaçlı araçlar kullanılmaz.

Hangi veriler işlenebilir?#

  • Sunucu kayıtları: Site GitHub Pages üzerinde barındırılmaktadır. Barındırma/CDN katmanı güvenlik ve performans amaçlarıyla teknik kayıtlar (IP adresi, istek zamanı, istenen URL, user-agent vb.) üretebilir.

Üçüncü taraf hizmetler#

  • Yorumlar (Giscus / GitHub Discussions): Yorumlar, Giscus üzerinden GitHub Discussions altyapısıyla sağlanır. Yorum bölümü yüklendiğinde Giscus ve GitHub altyapısına istek yapılır; yorum yazmak için GitHub hesabı ve yetkilendirme gerekebilir.
    • Giscus Gizlilik Politikası
    • GitHub Gizlilik Politikaları

Çerezler ve benzeri teknolojiler#

Bu sitede reklam/izleme amaçlı çerez (cookie) kullanılmaz.

  • Tercihler (tarayıcı depolaması): Tema gibi kullanıcı tercihleri tarayıcı depolamasında saklanabilir (ör. localStorage içinde pref-theme). Bu bilgiler cihazda tutulur ve site sunucusuna gönderilmez.
  • Yorumlar: Yorumlar yüklendiğinde Giscus/GitHub tarafında çerez/benzeri depolama mekanizmaları devreye girebilir.
